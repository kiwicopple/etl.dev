# 추출

## 소스 시스템에서 데이터 추출

데이터 추출은 ETL 프로세스의 첫 번째 단계로, 다양한 소스 시스템에서 데이터를 검색하는 과정입니다. 이 단계는 데이터 변환 및 로딩 단계를 위한 기초를 마련하는 중요한 단계입니다.

### 데이터 소스의 종류

데이터는 다음과 같은 다양한 소스에서 추출될 수 있습니다:

- **데이터베이스:** SQL, NoSQL 데이터베이스.
- **클라우드 스토리지:** AWS S3, Google Cloud Storage.
- **API:** RESTful 서비스, 웹 API.
- **파일 시스템:** CSV, Excel, XML 파일.
- **스트리밍 데이터:** IoT 디바이스, 소셜 미디어 등의 실시간 데이터.

### 데이터 추출 방법

추출 방법은 소스 유형과 데이터의 특성에 따라 다릅니다:

- **전체 추출:** 소스 시스템에서 모든 데이터를 가져오는 것.
- **증분 추출:** 마지막 추출 이후에 변경된 또는 추가된 데이터만 검색하는 것.
- **논리적 추출:** 논리적 뷰나 쿼리를 기반으로 데이터를 추출하는 것.
- **물리적 추출:** 데이터를 그대로 복사하는 것.

### 데이터 추출의 도전과 해결책

데이터 추출 과정에서 흔히 발생하는 도전과 해결책은 다음과 같습니다:

- **데이터 품질 문제:** 불일치하거나 누락된 데이터는 데이터 검증 규칙을 구현하여 해결할 수 있습니다.
- **복잡한 데이터 형식:** 변환 도구를 사용하여 데이터를 일관된 형식으로 변환할 수 있습니다.
- **대량의 데이터:** 대용량 데이터셋을 효율적으로 처리하기 위해 증분 추출 방법을 선택할 수 있습니다.
- **시스템 성능:** 추출 작업을 비저류 시간대에 예약함으로써 부하를 균형있게 분산할 수 있습니다.
