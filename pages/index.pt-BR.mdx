# ETL: Extrair, Transformar, Carregar

Na computação, extrair, transformar, carregar é um processo em três fases onde os dados são extraídos de uma fonte, transformados e carregados em um novo destino.

## O que é ETL (Extrair, Transformar, Carregar)?

ETL significa Extrair, Transformar, Carregar. É um processo usado em armazenamento de dados que envolve:

- **Extrair** dados de vários sistemas de origem.
- **Transformar** os dados em um formato adequado para análise.
- **Carregar** em um banco de dados de destino final, data warehouse ou data lake.

## Importância do ETL no Processamento de Dados

ETL é vital para tomada de decisões baseada em dados, pois:

- Consolida dados diversos em um formato unificado.
- Garante qualidade e precisão dos dados.
- Facilita o armazenamento e análise eficientes dos dados.

## Visão Geral do Fluxo de Trabalho em ETL

O fluxo de trabalho típico em ETL consiste em três etapas:

1. **Extrair:** Os dados são coletados de múltiplos sistemas de origem, que podem variar em formato e estrutura.
2. **Transformar:** Os dados extraídos passam por limpeza, validação e conversão para se ajustarem ao esquema de destino.
3. **Carregar:** Os dados transformados são transferidos para o data warehouse para armazenamento e análise.

## Conceitos Chave em ETL

Entender esses conceitos chave é essencial em ETL:

- **Integração de Dados:** Combinação de dados de diferentes fontes em uma visão única e coesa.
- **Limpeza de Dados:** Melhorar a qualidade dos dados corrigindo imprecisões.
- **Transformação de Dados:** Conversão de dados de um formato ou estrutura para outro.
- **Processamento em Lotes vs. Processamento em Tempo Real:** Decidir entre processar dados em grandes lotes ou de forma contínua e em tempo real.
