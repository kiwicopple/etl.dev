# Ekstraksi

## Ekstraksi Data dari Sistem Sumber

Ekstraksi data adalah tahap pertama dari proses ETL, melibatkan pengambilan data dari berbagai sistem sumber. Langkah ini sangat penting karena menetapkan dasar bagi tahap transformasi data dan pemuatan data.

### Jenis Sumber Data

Data dapat diekstraksi dari berbagai sumber, termasuk:

- **Database:** SQL, database NoSQL.
- **Penyimpanan Awan:** AWS S3, Google Cloud Storage.
- **API:** Layanan RESTful, API web.
- **Sistem Berkas:** Berkas CSV, Excel, XML.
- **Data Streaming:** Data real-time dari perangkat IoT, media sosial.

### Metode Ekstraksi Data

Metode ekstraksi tergantung pada jenis sumber dan sifat data:

- **Ekstraksi Penuh:** Mengambil semua data dari sistem sumber.
- **Ekstraksi Incremental:** Mengambil hanya data baru atau yang berubah sejak ekstraksi terakhir.
- **Ekstraksi Logis:** Mengambil data berdasarkan tampilan logis atau kueri.
- **Ekstraksi Fisik:** Mengkopi data tepat seperti yang disimpan.

### Tantangan dan Solusi Ekstraksi Data

Tantangan umum dalam ekstraksi data termasuk:

- **Masalah Kualitas Data:** Data yang tidak konsisten atau hilang dapat diatasi dengan menerapkan aturan validasi data.
- **Format Data yang Rumit:** Gunakan alat transformasi untuk mengonversi data menjadi format yang seragam.
- **Volume Besar:** Pilih metode ekstraksi inkremental untuk mengatasi kumpulan data besar dengan efisien.
- **Kinerja Sistem:** Seimbangkan beban dengan menjadwalkan ekstraksi selama jam-jam sibuk rendah.
