# Mengubah

## Mengubah Data dalam ETL

Dalam fase pengubah, data yang diekstraksi dari sistem sumber dibersihkan, didstandarisasi, dan diformat ulang untuk toko data target. Langkah ini kritis untuk memastikan keandalan dan kegunaan data.

### Pembersihan dan Validasi Data

Memastikan kualitas data melibatkan:

- **Pembersihan Data:** Menghapus ketidakakuratan, duplikat, dan memperbaiki kesalahan.
- **Validasi Data:** Memastikan data memenuhi kriteria dan standar tertentu.

### Teknik Transformasi Data

Teknik kunci dalam transformasi data meliputi:

- **Normalisasi:** Membentuk data untuk mengurangi redundansi.
- **Pemetaan Data:** Melakukan penyesuaian elemen data dari sumber ke format tujuan.
- **Agregasi:** Menyusun data terperinci untuk analisis.
- **Pengayaan Data:** Memperkaya data dengan informasi tambahan dari sumber eksternal.

### Praktik Terbaik Transformasi Data ETL

Untuk memastikan transformasi data yang efektif:

- **Tentukan Aturan yang Jelas:** Tetapkan prosedur standar untuk menangani jenis data yang berbeda.
- **Otomatisasi Proses:** Manfaatkan alat ETL untuk mengotomatiskan tugas yang berulang.
- **Pantau Kualitas Data:** Secara rutin periksa data untuk kesalahan dan inkonsistensi.
- **Pendekatan Berulang:** Terus sempurnakan proses transformasi berdasarkan umpan balik dan perubahan data.
